<h4>Folder structure</h4>
Workspace apps:
- web
- mobile
- backend

Libs are created as `/libs/[scope]/[type]-[name]` where:
- scope is `web, mobile, backend, shared` 
- type is `feature, utils, data-access, ui`

Considering this example libs are:
 - web
   - feature-products
   - data-access-products
 - backend
   - features-products
 - shared
   - features-products
